Проектирование архитектуры будущей базы данных

С использованием инструмента для построения диаграмм Lucidchart, я создала нормализованную структуру, которая улучшит организацию и доступность данных о кланах в игре. Данная структура базы данных состоит из шести таблиц, каждая из которых имеет четко определенную смысловую границу:

- Информация о клане
- Достижения клана
- Участники клана
- Требуемые условия для вступления в клан
- Столица клана
- Войны
